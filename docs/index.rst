ml-hpi Documentation
====================

The **Multi-Language Hierarchical Programming Interface** (ml-hpi) defines a
declarative, language-neutral specification for verification APIs. From a single
source — authored in YAML/JSON or derived from an existing SystemVerilog
``interface class`` — ml-hpi generates correct, idiomatic bindings for
SystemVerilog, C++, Python, C, and PSS simultaneously.
